You'll need a few things:

• A Pokemon Red ROM
• A Hex editor - I'll be using Translhextion.

Our Goal is Simple and Just:

We want to make a Pokemon hack where Brock wants to fuck us instead of fight us with his rock hard onix.



Open your Pokemon Red ROM using your hex editor. You'll see a lot of hex code on the left,
some junk on the right. If you highlight some hex, you'll see where and how it's appearing in
the right column. It is a currently illegible translation of the hex code.

It should say POKEMON RED in there.


We need to be able to see the in-game text within this unreadable right column if we want to make it better.



Using Search/Relative Search, you can find within the code certain patterns that you know
exist. We're looking to edit the word "fight," so use the Relative Search to locate where it
could be in the code. In this initial step, it might be best to use a very particular
word (such as Charmander) to figure out what some of the alphabetical letters
are within the code.

After searching, I'm left with the hex code A5, A8, A6, A7, B3.

If you open your Table Manager, there's a grid that has values for any hex value.

Put "f" in the A5 slot, and you'll figure out that A8 ends up where "i" would be (moving left
to right, bottom to top). Fill in the entire alphabet this way. Since Pokemon Red has
capital and lower case letters, we can figure out where capital letter values are
stored as well (try searching some names or whatever).


You'll end with a table that looks kind of like this one. Save your .tbl file.

This site has a bunch of useful info and even a table guide for Pokemon Red and Blue.

Back in your hex editor, apply the table file.
For Translhextion, check the Thingy View box to view the right column with your table applied.

There are several times the letter pattern "fight" appears in the game, so we're going to
search and change them all. The text is not going to be at all legible in the way it is when
displayed in the game, and so sentences and phrases in the code may not be connected
as they are in-game. An OP hacker uses wits and skills to get around this,
though it's far more erotic to replace all fighting with fucking.

Since "fight" is five letters long and "fuck" is only four, let's add an extra "!" to it for
emphasis. You could make it a blank space or whatever, but this is the coolest way to do it.

Translhextion has a feature to find and replace all of a certain pattern, so let's replace all instances of "fight" with "fuck!"

Save your ROM.

Open it in BGB - you'll get an error message you can ignore. (A Real Gameboy wouldn't care, okay)
